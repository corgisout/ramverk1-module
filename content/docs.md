###Guide till JSON API i för väder hämtning med ip eller plats

APIet tar emot två argument: `search` och `when`.

För att hämta väder data måste båda argumenten skickas med.

* plats eller IP
* past eller future


Tillbaka ska du få ett objekt med vädret antingen denna veckas eller senaste månadens

ett svar kan se ut såhär om man skickar

apiet kollar först om du har skickat en plats eller en IP och sen omvandlar den platsen till
longitude och latitude. detta med hjälp av locationiq eller ipstack. lon och lat skickas sedan i
med en GET request till ett annat api darksky för att få fram vädret för platsen.
```
https://api.darksky.net/forecast/{apiKey}/{$lon},{$lat},{$dates[$i]}?lang=sv&units=si
```
den färdiga requesten kan ser ut såhär om du vill ha veckans väder i karlskrona.
```
/redovisa/htdocs/api?search=karlskrona&when=future&submit=create+json
```
